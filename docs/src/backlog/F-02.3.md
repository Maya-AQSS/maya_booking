# [BACKEND] F-02.3 — Bloqueos Anidados de Espacio y Puesto

## 🏷️ Capa Técnica
**Capa:** `backend`
**Impacta en:** Equipo de Desarrollo Odoo

**Prioridad:** `Must`

---

**Descripción:**
Como Profesor, quiero que al reservar una sala entera se bloqueen sus puestos, o al reservar un puesto la sala quede parcialmente ocupada.

**Criterios de Aceptación (Checklist):**
- [ ] **Escenario 1 (Happy Path):** Reserva de espacio padre oculta sus hijos.
- [ ] **Escenario 2 (Edge Case):** Reserva de un hijo marca al padre como no disponible para uso total.
- [ ] **Requisito Técnico:** Restricción de base de datos o lógica server-side gestionada vía ORM/Vistas en Odoo.
