# [FRONTEND] F-03.1 — Vista Semanal y Cuadrícula de Ocupación

## 🏷️ Capa Técnica
**Capa:** `frontend`
**Impacta en:** Equipo de Desarrollo Odoo

**Prioridad:** `Must`

---

**Descripción:**
Como usuario, quiero ver un calendario semanal con la ocupación gráfica de toda una categoría.

**Criterios de Aceptación (Checklist):**
- [ ] **Escenario 1 (Happy Path):** Bloques pintados por estado en la vista calendario.
- [ ] **Escenario 2 (Edge Case):** Clicar bloque muestra quién lo reservó y el motivo.
- [ ] **Requisito Técnico:** Restricción de base de datos o lógica server-side gestionada vía ORM/Vistas en Odoo.
