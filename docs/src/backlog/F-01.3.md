# [FULLSTACK] F-01.3 — Gestión de Puestos de Trabajo

## 🏷️ Capa Técnica
**Capa:** `fullstack`
**Impacta en:** Equipo de Desarrollo Odoo

**Prioridad:** `Should`

---

**Descripción:**
Como Vicedirección, quiero gestionar puestos individuales dentro de espacios para permitir reservas parciales.

**Criterios de Aceptación (Checklist):**
- [ ] **Escenario 1 (Happy Path):** Un puesto se asigna a un espacio padre.
- [ ] **Escenario 2 (Edge Case):** El check 'reservable directamente' oculta o muestra el puesto en las búsquedas.
- [ ] **Requisito Técnico:** Restricción de base de datos o lógica server-side gestionada vía ORM/Vistas en Odoo.
