# [BACKEND] F-01.4 — Integración con Módulo de Inventario

## 🏷️ Capa Técnica
**Capa:** `backend`
**Impacta en:** Equipo de Desarrollo Odoo

**Prioridad:** `Must`

---

**Descripción:**
Como Vicedirección, quiero que los elementos prestables provengan del módulo de Inventario automáticamente.

**Criterios de Aceptación (Checklist):**
- [ ] **Escenario 1 (Happy Path):** Sincronización automática de elementos 'Prestables'.
- [ ] **Escenario 2 (Edge Case):** Si un objeto pierde la propiedad prestable, deja de ser reservable.
- [ ] **Requisito Técnico:** Restricción de base de datos o lógica server-side gestionada vía ORM/Vistas en Odoo.
