# [FULLSTACK] F-03.2 — Generación y Consulta Vía QR

## 🏷️ Capa Técnica
**Capa:** `fullstack`
**Impacta en:** Equipo de Desarrollo Odoo

**Prioridad:** `Could`

---

**Descripción:**
Como centro, quiero generar códigos QR estáticos en las puertas para que con un móvil se vea su estado de hoy.

**Criterios de Aceptación (Checklist):**
- [ ] **Escenario 1 (Happy Path):** QR dirigiendo a una URL pública de Odoo para el recurso.
- [ ] **Escenario 2 (Edge Case):** La página sólo muestra la info de ocupación de hoy.
- [ ] **Requisito Técnico:** Restricción de base de datos o lógica server-side gestionada vía ORM/Vistas en Odoo.
